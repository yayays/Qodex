# Qodex v0.1.2

## Highlights

- Added built-in WeChat support through `builtin:wechat-openclaw-compat`
- Added Tencent WeChat QR-login adapter with token, sync buffer, and context-token persistence
- Added WeChat quick-start flow that generates config, starts Qodex, prints a QR link, and verifies saved login state
- Added visible channel status transition logs so QR login progress is observable during startup

## Included

- Built-in WeChat channel plugin and Tencent transport adapter
- WeChat inbound text polling and outbound text replies
- `quick:start` support for `wechat` and `qq`
- README updates documenting WeChat support and quick-start usage
- Tests covering WeChat QR login, timeout recovery, inbound polling, outbound reply context tokens, and startup status visibility

## Notes

- WeChat v1 currently supports QR login and basic text messaging
- Media send and receive are not included in this release
- `quick:start` now exposes `wechat` and `qq` as the user-facing channel choices
